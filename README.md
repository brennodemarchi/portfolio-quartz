# My Portfolio and Digital Garden 🌱

This repository is a space where I share my projects, thoughts, and learnings, built with [Quartz]([https://github.com/jackyzha0/quartz]), an incredible tool developed by [Jacky Zhao](https://quartz.jzhao.xyz).

- **Interface**: Minimalist and responsive design for different devices.
